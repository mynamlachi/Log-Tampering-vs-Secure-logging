# 🔐 Log Tampering Attack vs Secure Logging Defense Simulator

A GUI-based Cyber Range simulation developed in Python that demonstrates how attackers tamper with system logs and how secure logging mechanisms detect such attacks using SHA-256 hashing.

---

## 📌 Project Overview

In cybersecurity, logs are crucial for monitoring and forensic analysis. Attackers often modify logs to hide their malicious activities. This project simulates such a scenario and provides a defense mechanism using cryptographic hashing to ensure log integrity.

---

## 🎯 Objectives

- Simulate a log tampering attack  
- Implement secure logging using SHA-256 hashing  
- Detect unauthorized modifications in logs  
- Provide a user-friendly GUI  
- Demonstrate cybersecurity concepts  

---

## ⚙️ Technologies Used

- Python  
- Tkinter  
- hashlib (SHA-256)  
- File Handling  

---

## 🧠 How It Works

1. Generate logs  
2. Create hash  
3. Tamper logs  
4. Recalculate hash  
5. Detect mismatch  

---

## 🚀 Features

- Log generation  
- Attack simulation  
- Tampering detection  
- GUI interface
